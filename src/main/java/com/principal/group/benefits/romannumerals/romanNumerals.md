# Roman Numerals

## Problem description

The goal of this kata is to convert between Arabic numbers and Roman numerals.  

Info about Roman numerals: http://www.novaroma.org/via_romana/numbers.html

###Table:

| Symbol  | Value    |
| ------- | -------- |
| I       |    1     |
| V       |    5     |
| X       |   10     |
| L       |   50     |
| C       |  100     |
| D       |  500     |
| M       | 1000     |


**Example**

1994 -> MCMXCIV

because 1000=M 900=CM 90=XC 4 = IV
